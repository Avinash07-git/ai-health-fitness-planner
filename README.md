# AI Health & Fitness Planner

An AI-powered Streamlit application that generates personalized diet and fitness plans based on a user's profile, goals, activity level, and dietary preferences.

## Features
- Personalized dietary plan
- Personalized fitness plan
- Gemini API integration
- Interactive Streamlit UI
- Goal-based health recommendations

## Tech Stack
- Python
- Streamlit
- Agno
- Google Gemini API

## How to Run
1. Clone the repository
2. Create and activate a virtual environment
3. Install dependencies using:
   `pip install -r requirements.txt`
4. Run the app using:
   `streamlit run app.py`

## Notes
- Add your Gemini API key in the app sidebar when prompted
- Sensitive/local files like `.env` and `.venv` are excluded from version control
